Intruder Detection System

A Project related to security of house from intruders

In Intruder detection and automatic email alerting system, first we capture the image of person that enters the house by using the webcam.

The general idea for the project is to capture intruders who try to break through houses with the help of modern technology such as machine learning and deep learning. Our objective is to provide automation in this process, so it doesn't matter where the user is, he can see all the people coming inside his house with or without permission. He can check all images stored in database and can identify any suspicious activity and can report it.

FINISH

User will verify the image and will take the decision accordingly...

Now the system will check whether the person is family member or not by using Machine learning and Deep Learning concepts. We are storing the images of family members in the database for verification and comparing the image of the person who enter the house. For the entire project we are using Python programming language and Open CV library. We are using encoding and decoding model of communication for formatting the data. After the verification of image, the system will send the image in specific format to the family member.
